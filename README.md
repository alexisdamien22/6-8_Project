# 6-8_Project

## Migration vers Node.js

1. Installer les dépendances :

```bash
npm install
```

2. Démarrer le serveur Node.js :

```bash
npm start
```

3. Ouvrir le navigateur sur :

```
http://localhost:3000
```

4. Le fichier de frontend principal est `index.html`.
5. Le backend est dans `server.js`, il sert les fichiers statiques et contient l'endpoint d'exemple `/api/status`.
